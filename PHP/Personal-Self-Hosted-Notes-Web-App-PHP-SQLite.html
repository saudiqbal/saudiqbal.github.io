Personal Self Hosted Notes Web App Using PHP, SQLite and TinyMCE

A quick and easy to use self hosted Notes web app using WYSIWYG TinyMCE editor. Just upload all the files in the folder to your web server and it is ready for use. Includes syntax highlighting and notes search.
